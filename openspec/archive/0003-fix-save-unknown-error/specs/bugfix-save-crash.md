# Specs: 相片保存时主线程安全性约束

## 用户故事
作为长截图的使用者，在我由于拼接成功想要保存结果到了最后一步时，我不希望看到“未知错误”的保存失败提示，我期望顺畅地看到“保存成功”。

## 边界与约束
1. **主线程安全性**：`UIImageWriteToSavedPhotosAlbum` 调用必须在 `Main Thread` 中发起。
2. **多线程权限回调**：相册请求权限接口 `PHPhotoLibrary.requestAuthorization` 的回调不保证在哪个线程中执行。

## 验收条件
- 验证当第一次运行应用，弹出“需要相册权限”确认后，照片能否成功保存成功。
- 验证当已授权用户点击保存时，是否正常且毫无错误地保存了照片。
