# Specs: Markup 缩放边界一致性修复

## 场景

用户在长截图编辑页进行涂鸦/打码时，需要看到底图和笔迹层在同一套缩放边界内工作，避免到达最小值或最大值时两层表现不一致。

## 需求

1. 编辑器必须为底图层和 `PKCanvasView` 生成同一组 `minimumZoomScale` 与 `maximumZoomScale`。
2. 每次布局更新后，底图层和 `PKCanvasView` 的缩放边界必须重新同步。
3. 当前缩放值必须被钳制到统一边界内，再同步到底图层与笔迹层。

## 边界

1. 当长图高度超过可视区域时，最小缩放系数必须允许用户完整查看长图。
2. 当图片较短时，最小缩放系数不得超过 `1.0`。
3. 本次修复不改变最大缩放上限 `4.0`。
