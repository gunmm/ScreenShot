# Specs: Markup 缩放边界禁用回弹

## 场景

用户在长截图编辑页双指缩放到最小或最大边界后，希望继续捏合时不再出现弹性回弹，而是直接停在边界值。

## 需求

1. 编辑器到达 `minimumZoomScale` 后，继续双指缩小必须不产生回弹。
2. 编辑器到达 `maximumZoomScale` 后，继续双指放大必须不产生回弹。
3. 底图层与 `PKCanvasView` 在缩放边界处的交互反馈必须一致。

## 边界

1. 本次变更不修改 `minimumZoomScale` 与 `maximumZoomScale` 的计算方式。
2. 本次变更不修改滚动偏移同步逻辑。
3. 本次变更仅约束缩放边界处的捏合反馈。
