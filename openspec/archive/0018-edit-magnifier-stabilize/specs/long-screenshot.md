# Specs: long-screenshot

## 新增需求：编辑放大镜稳定性修正

### 场景
用户在编辑页拖动裁剪边界时，希望放大镜不仅固定在顶部角落，还能稳定显示接缝局部内容，不随界面元素一起抖动。

### 要求

- 放大镜应优先基于原始图片像素生成局部预览，而不是对整个编辑区做截图。
- 放大镜在拖动过程中不应因边框、手柄或布局刷新而出现明显抖动。
- 放大镜尺寸应为 160 x 160。
- 左上角或右上角固定定位规则保持不变。

### 验收

- 拖动顶部或底部手柄时，放大镜画面比上一版更稳定。
- 放大镜显示尺寸为 160 方形。
- 放大镜仍只出现在左上角或右上角。
