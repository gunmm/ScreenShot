# Specs: long-screenshot

## 新增需求：编辑放大镜显示真实接缝内容

### 场景
用户在编辑页拖动裁剪边界时，希望放大镜显示的就是当前接缝上下两侧的真实内容，而不是只来自某一张分片的局部裁切。

### 要求

- 放大镜必须围绕当前被调整的接缝生成预览，并同时展示接缝上下两侧的分片内容。
- 顶部手柄应反映上一分片底部与当前分片顶部的接缝。
- 底部手柄应反映当前分片底部与下一分片顶部的接缝。
- 没有相邻分片时，放大镜应优雅退化，但仍保持 seam 语义和稳定性。

### 验收

- 放大镜内容与当前手柄对应的真实接缝一致。
- 放大镜不会恢复明显抖动。
- 160 方形尺寸与左上/右上角固定位置保持不变。
