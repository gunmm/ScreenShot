# Specs: long-screenshot

## 新增需求：编辑拖动放大镜预览

### 场景
用户在 EditViewController 中拖动分片顶部或底部手柄微调裁剪边界时，需要看清接缝处的像素细节，以便更准确地判断最佳衔接位置。

### 要求

- 系统必须在用户开始拖动裁剪手柄时显示局部放大镜预览。
- 放大镜必须围绕当前被拖动的裁剪边界展示接缝附近内容，并随拖动实时更新。
- 放大镜应默认避开手指和手柄本体，优先显示在拖动点上方；若空间不足，可调整到下方。
- 用户结束、取消或中断拖动后，放大镜必须立即隐藏。
- 放大镜是编辑过程中的辅助反馈，不得改变当前裁剪区间计算与 Done 回传结果。

### 验收

- 顶部手柄和底部手柄拖动路径都能触发放大镜。
- 放大镜内容会跟随当前裁剪边界同步变化，而不是停留在拖动开始时的静态图像。
- 放大镜消失后，编辑页恢复当前正常视图，不残留遮挡层。
