# 3 分钟演示脚本（口播 + 操作）

> 目标：在 3 分钟内展示“跨 App 长内容 → 一键生成可分享长图”的生产力价值与差异化亮点。

## 0:00 - 0:20（痛点与定位）

口播：

- “很多信息是**长滚动内容**：客服对话、订单详情、攻略/评论流。海外团队协作时最常见的需求是：**一张图讲清楚**，方便分享与留证。”
- “但系统自带的整页截图覆盖范围有限，跨 App 场景做不到。”

## 0:20 - 1:40（主演示：跨 App 录制 → 生成）

操作：

1. 打开 LongScreenShot，展示引导卡片（如何开始）
2. 点击系统录制入口开始录制
3. 切到目标 App（建议选：聊天/订单/评论流），连续向下滚动 10-20 秒
4. 回到 LongScreenShot，点击“一键生成长截图”

口播（生成时）：

- “我们会自动筛选有效下滑帧，减少重复与抖动，然后用 OpenCV 做重叠检测与裁剪拼接。”

## 1:40 - 2:20（结果惊艳点：分享导出 / 更干净）

操作：

- 生成成功后展示长图预览
- 点击“分享”，打开系统分享面板（选择：保存到相册/发到聊天工具）

口播：

- “这张长图可以直接分享给同事/客户，或保存留证。”
- “整个过程在本地完成，不上传内容，隐私更友好。”

## 2:20 - 2:50（差异化总结）

口播（3 点）：

- 跨 App 捕获
- 智能筛帧（只保留有效滚动）
- 重叠检测裁剪拼接（减少重复，更连续）

## 2:50 - 3:00（商业化与路线图一句话）

口播：

- “商业化可以走 Freemium：免费核心生成，Pro 提供高清导出、批量与历史管理、自动裁边/去重增强等。”
