# Specs: long-screenshot

## 新增需求：马赛克页支持真实像素化编辑

### 场景

当前马赛克页仍是占位骨架。系统应允许用户直接在长图上滑动，对局部区域应用真实像素化马赛克，并把结果图回传到主页。

### 要求

- MosaicViewController 必须继续接收一张长图作为输入，并在点击完成后输出一张编辑后的长图。
- 页面必须支持长图缩放与浏览，便于用户在局部位置精修马赛克区域。
- 用户单指在图片上滑动时，系统必须在对应位置实时显示像素化效果。
- 页面必须提供可调的粗细和透明度控件；新笔划应按当前参数生效。
- 粗细和透明度控件当前值必须以 0-100% 的方式展示，避免直接暴露内部 pt 数值。
- 页面必须支持撤销和重做最近的马赛克笔划。

### 验收

- 马赛克页可以对长图进行真实像素化编辑，而不是仅展示占位说明。
- 完成后的结果图会回传主页，并继续沿用现有展示和导出链路。
- 撤销、重做后的预览结果与最终导出结果一致。
