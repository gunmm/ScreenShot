# Specs: 涂鸦编辑器缩放精修增量

## 场景
用户在长截图预览页进入“涂鸦/打码”后，需要能够放大某一局部区域进行精细遮挡，并确保笔迹在任意缩放级别下都与底图保持对齐。

## 需求
1. 编辑器必须提供单一内容坐标系，底图和笔迹共享同一个内容尺寸。
2. 编辑器必须支持双指缩放与拖动长图。
3. 在最大缩放范围内，用户可以针对头像、昵称、手机号等小区域进行局部绘制。
4. 缩放和平移只影响观察视图，不改变导出时的笔迹几何关系。
5. 点击“完成”后，导出的图片应以原图分辨率输出，笔迹位置与编辑时看到的位置一致。

## 边界
1. 当图片宽度与屏幕宽度一致时，初始缩放应完整铺满可用宽度。
2. 当缩放后内容宽度小于可视区域宽度时，内容应居中显示，避免横向贴边。
3. 本次变更不要求跨会话恢复未完成的笔迹工程。
