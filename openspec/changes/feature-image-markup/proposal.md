# Proposal: 添加长截图涂鸦打码功能 (PencilKit Zoomable Markup)

## 1. 动机 (Motivation)
长截屏的常见使用场景是分享聊天记录，用户在此场景下有极强的“隐私保护”需求（打码头像、昵称）。当前 App 生成长图后，仅支持调整拼接位置，缺少图片编辑功能，导致用户需要借助第三方 App 甚至微信自带编辑功能去打码，体验割裂。引入原生的 PencilKit 涂抹画笔，能在 App 内完成闭环，极大提升工具的留存率和好评率。

## 2. 范围 (Scope)
- 在生成的长截图预览页底部操作栏中，增加“涂鸦/打码”入口。
- 将原本的“编辑”按钮更名为更直观的“拼接调整”。
- 提供一个全屏的图片编辑页面（基于 iOS 原生 `PencilKit` 和 `PKToolPicker`）。
- 支持用户使用马克笔等工具进行实色涂抹遮盖。
- 支持长图缩放和平移，以便局部放大精修。
- 支持保存修改并覆盖当前预览的长图。

## 3. 非目标 (Non-Goals)
- 暂时不实现复杂的像素化马赛克效果（后续迭代再考虑）。
- 暂时不实现基于原图的分段涂鸦（避免破坏拼接算法的特征点）。
- 暂时不实现插入文本、贴纸等复杂编辑功能。
- 暂时不实现涂鸦工程文件持久化与再次编辑恢复。

## 4. 验收标准 (Acceptance Criteria)
1. 拼接完成后，能在主页底部栏看到“涂鸦/打码”按钮。
2. 点击“涂鸦”进入全屏编辑界面，可以通过手指/Apple Pencil 在图片上画画。
3. 界面会弹出系统原生的画笔选择器（包含颜色、笔触大小）。
4. 用户可以双指缩放和拖动长图，在放大状态下进行局部精修时，笔迹与底图保持稳定对齐。
5. 点击“完成”后，涂鸦内容会合并到主页的图片中，并且可以正常保存到相册或分享 PDF，免费版的去水印控制依然生效。
